This document gives you general information on this release of Aladdin StuffIt Engine SDK 5.1 for Windows, Linux and Solaris.

Welcome!

This SDK release for Windows, Linux and Solaris is a public beta version. It includes static and dynamic libraries and samples. It works with the Visual C++ 6.0 compiler on Win32 systems (Windows 95/98/ME, Windows NT, Windows 2000) and egcs 1.1/gcc 2.95 on Linux and Solaris.

Installation

Install by expanding the stuffitengine-5.*.zip or stuffitengine-5.*.tar.gz file into a directory of your choice.

This SDK contains the following directories:

Directory Windows Linux Solaris Contents
doc/ Documentation for StuffIt Engine and test applications.
src/ Headers and sample sources.
linux/, linux/make/ Linux sample projects.
linux/bin/ Linux sample executables.
linux/lib/ Linux static and shared library.
solaris/, solaris/make/ Solaris sample projects.
solaris/bin/ Solaris sample executables.
solaris/lib/ Solaris static and shared library.
win32/, win32/project/ Win32 sample projects.
win32/bin/ Win32 static and shared library and sample executables.
win32/binS/ Win32 static and shared library and sample executables, statically linked with C and C++ runtime libraries.
win32/binT/ Win32 self-extracting executables.
