MacBinary file information

Documentation

MacBinary file information. This corresponds to the file information section (header) of a MacBinary file.

This code is never returned by the classifier and does not correspond to a separate format. It is used to designate a part of a MacBinary file processed by the engine, when it may be necessary to distinguish it from other parts in an event callback.
