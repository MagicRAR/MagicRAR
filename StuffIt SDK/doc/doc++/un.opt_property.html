Documentation

An optional property that may or may not be available at different times. Equivalent to a un::property with an additional bool value to indicate availability.
opt_property()
Default constructor

opt_property(const value_type& value)
Conversion constructor. Initializes the value.
Parameters:
t - initial value of the property

~opt_property()
Destructor

value_type& operator)()
Accessor (get). Has no arguments, and returns the reference to the property, as held by the variable. Marks the property as available (assumes that non-const access assigns/modifies the previously unavailable property). Obviously, this works best with const-correct code.
Returns:
non-const reference to the property value

const value_type& operator)() const
Const accessor (get). Has no arguments, and returns the const reference to the property, as held by the variable.
Returns:
const reference to the property value

void operator)(const value_type& value)
Mutator (set). Returns nothing, and has one argument - the new value of the property - which it assigns to the variable. Marks the property as available.
Parameters:
t - new value of the property
