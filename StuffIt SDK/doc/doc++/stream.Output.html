Documentation

This abstract class is the superclass of all classes representing an output stream of characters.

This class may serve as an abstract interface for objects that provide externally driven output via write().

Applications that need to define a subclass of stream::Output must always provide a method that writes one character to the output.

virtual void resize(stream::size size )
Resizes the stream. Truncates or extends the container object (such as file) of a subclass. The contents of the extended portion are undefined. When truncating, this method can truncate the container object only at the current stream position; it cannot shorten the container object.
Throws:
