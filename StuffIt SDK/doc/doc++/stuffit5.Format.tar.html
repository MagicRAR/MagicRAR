Unix tar archive format

Unix tar archive format. Unix tar: tape archive. Used as an uncompressed archive format that combines multiple files and folder (directory) hierarchies into a single file, not necessarily (rarely, in fact) for writing to a tape. Often combined with compression by Unix compress, gzip, and lately bzip2.
