Determines if a writer is capable of creating a format.

Returns true when the writer is capable of creating the format described by the format parameter, false otherwise. The return value indicates that the writer this function applies to includes code required to create the format.

Returns:
true if the writer can create the format
Parameters:
format - the archive format
writer - the writer reference
