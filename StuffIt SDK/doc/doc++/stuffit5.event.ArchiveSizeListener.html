Event handler for archive size.

This event callback is invoked when a writer needs to determine the maximum size of the next archive segment of a multi-segment archive.

Use stuffit5::Writer::archiveSize() or getWriterArchiveSize)() and setWriterArchiveSize)() to obtain the proposed size of the next segment the writer is going to try to create, and to change this size, respectively.
