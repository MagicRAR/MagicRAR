Documentation

This class generates event notifications for ArchiveInfoEvent. It maintains a list of ArchiveInfoListeners.

The implementation of this class makes a copy of the listener list when firing events. This allows listeners to be safely added and removed during event handling but it may lead to performance problems in environments where events are used frequently.
