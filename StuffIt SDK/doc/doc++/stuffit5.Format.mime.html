MIME encoded message (file) format

Documentation

MIME encoded message (file) format. MIME stands for Multipurpose Internet Mail Extensions. A text-based 7-bit email message format with typed (and possibly named) sections and attachments. More flexible than any other 7-bit encoding. Many variations (sometimes buggy and non-compliant) despite the existence of an official RFC.
