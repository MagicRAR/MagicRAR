The first segment of a segmented StuffIt archive format

Documentation

The first segment of a segmented StuffIt archive format. Used in Aladdin Systems products. Returned by the classifier when it determines that a file is the first segment of a series of StuffIt archive segments. Expansion of a segmented StuffIt archive always begins with the first segment.
