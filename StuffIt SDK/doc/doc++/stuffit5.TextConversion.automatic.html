const stuffit5::TextConversion::type stuffit5::TextConversion::automatic

Text files are converted

Documentation

Text files are converted. Text conversion is applied to output files that are created by engine readers during decompression or decoding and are known to contain text. End-of-lines are changed according to a stuffit5::TextType.

Files that are known to contain text are defined as follows: either the format being decompressed or decoded by the reader contains an indication or flag that a file is a text file, or a file is a Macintosh file that has Macintosh file type of "TEXT" as its attribute stored in the archive.
