Documentation

This abstract class is the superclass of all classes representing an input stream of characters.

This class may serve as an abstract interface for objects that provide externally driven input via read().

Applications that need to define a subclass of stream::Input must always provide a method that returns the next character of input.

virtual stream::size skip(stream::size size = 1)
Skips over and discards size characters of data from the stream. This method may skip over a smaller number of characters, possibly 0. The actual number of characters skipped is returned.

This method reads data into an array until size characters have been read or the end of the stream has been reached. Subclasses are encouraged to provide a more efficient implementation of this method.

Returns:
the actual number of characters skipped
Parameters:
size - the number of characters to skip
